# 🐍 Snake Game (HTML / CSS / JavaScript — Single File)

이 프로젝트는 **HTML 파일 한 개** 안에 **CSS와 JavaScript가 모두 포함된 스네이크 게임**입니다.  
브라우저만 있으면 어디서든 실행 가능하며, 설치가 필요 없습니다.

---

## 🎮 게임 특징

- **단일 HTML 파일**로 구성 (CSS + JS 포함)
- **키보드 화살표 키로 조작**
  - ↑ ↓ ← → : 이동  
  - Space : 일시정지 / 재개  
  - R : 재시작
- 고급 UI 디자인 + 부드러운 애니메이션
- 로컬 스토리지 기반 **최고점(High Score) 자동 저장**
- 먹이를 먹을수록 **속도 증가**
- HUD(점수/컨트롤 패널) 제공

---

## 🛠 실행 방법

1. 저장소를 다운로드하거나 클론합니다.
2. 프로젝트 폴더 안의 `snake.html` 파일을 열어 실행합니다.
3. 웹 브라우저에서 자동으로 게임이 시작됩니다.

git clone https://github.com/your-repo/snake-game.git
